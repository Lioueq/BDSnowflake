# BigDataSpark

Анализ больших данных - лабораторная работа №1 - нормализация данных в снежинку

## Цель работы

Необходимо данные источника (файлы mock_data.csv с номерами), которые представляют информацию о покупателях, продавцах, поставщиках, магазинах, товарах для домашних питомцев трансформировать в модель снежинка/звезда (факты и измерения с нормализацией).

## Что сделано

1. Настроен сервис `postgres`
2. Реализован ddl-скрипт `00_create_staging.sql`, в котором описана структура таблицы, в которой будут лежать все исходные данные из всех csv файлов
3. Реализован `01_import.sh`, который импортирует все csv файлы в базу данных
4. Реализован ddl-скрипт `03_ddl.sql`, в котором описана структура таблиц фактов и измерений
5. Реализован dml-скрипт `04_dml.sql`, в котором происходит заполнение таблиц фактов и измерений из исходных данных.
